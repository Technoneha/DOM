# Greeting App

A simple interactive Greeting App built using HTML, CSS, and JavaScript.

## Features

- Header greeting message
- Four color boxes:
  - Red
  - Blue
  - Green
  - Yellow
- Input field to enter user name
- Greeting button
- Dynamic greeting update
- Clickable color boxes that change background color

## Technologies Used

- HTML
- CSS
- JavaScript

## Functionality

### Greeting Feature
When the user enters a name and clicks the button, the header changes from:

Hello, Username

Color Box Feature

When a box is clicked, its background changes to the respective color written inside the box.
